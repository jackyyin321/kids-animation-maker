# 🎬 儿童动画制作工具

一个专为6岁儿童设计的简单易用的网页动画制作工具。

## 功能特色

- 📸 **简单上传** - 拖放或点击上传照片
- 🎬 **可视化编辑** - 横向时间轴，直接拖动调整顺序
- ⏱️ **灵活时长控制** - 批量或单独调整每帧时长
- 📋 **轻松复制** - 一键复制帧
- 🎤 **音频功能** - 每帧可录音或上传音乐
- ▶️ **即时预览** - 大播放窗口实时查看效果
- 💾 **导出分享** - 保存项目、导出GIF

## 如何在Mac上使用

### 方法一：直接双击打开（最简单）

1. 将整个 `kids-animation-maker` 文件夹复制到Mac电脑
2. 双击 `index.html` 文件
3. 文件会在默认浏览器（Safari/Chrome）中打开
4. 开始创作！

### 方法二：使用浏览器打开

1. 打开Safari或Chrome浏览器
2. 按 `Command + O` 打开文件
3. 选择 `index.html` 文件
4. 开始使用

### 方法三：创建桌面快捷方式（推荐）

1. 右键点击 `index.html` 文件
2. 选择"制作替身"（Make Alias）
3. 将替身拖到桌面或应用程序文件夹
4. 重命名为"我的动画工作室"
5. 以后只需双击图标即可打开

### 方法四：添加到Dock（最方便）

1. 打开 `index.html` 文件
2. 在浏览器地址栏中，将网址拖到Dock
3. 创建一个书签快捷方式
4. 随时点击打开

## 文件传输方式

### 选项1：U盘/移动硬盘
1. 将 `kids-animation-maker` 文件夹复制到U盘
2. 插入Mac电脑
3. 复制到Mac的"文稿"或"下载"文件夹

### 选项2：AirDrop（如果两台都是Apple设备）
1. 在Windows电脑上压缩文件夹为ZIP
2. 通过iPhone/iPad中转
3. 使用AirDrop发送到Mac

### 选项3：云盘同步
1. 上传到iCloud Drive、Google Drive或Dropbox
2. 在Mac上下载

### 选项4：邮件发送
1. 压缩文件夹为ZIP（右键 → 发送到 → 压缩文件）
2. 发送邮件给自己
3. 在Mac上下载附件

### 选项5：局域网共享
1. 在Windows上启用文件共享
2. 在Mac的Finder中连接到Windows电脑
3. 复制文件夹

## 使用说明

### 快速开始
1. 上传照片（拖放或点击选择）
2. 拖动帧调整顺序
3. 点击播放按钮预览
4. 导出GIF保存作品

### 编辑帧
- 点击帧缩略图打开编辑器
- 调整时长滑块
- 录音或上传音频
- 点击"完成"保存

### 导出作品
- **保存项目**：保存为JSON文件，可以继续编辑
- **导出GIF**：生成动画GIF文件，可以分享

## 注意事项

### 录音功能
- 首次使用录音时，浏览器会请求麦克风权限
- 请点击"允许"以启用录音功能
- 如果使用Safari，可能需要在系统偏好设置中授权

### 文件大小
- 单张照片建议不超过5MB
- 总项目建议不超过50MB
- 照片会自动显示在画布上

### 保存位置
- 导出的GIF和项目文件会保存到"下载"文件夹
- 可以在浏览器的下载管理器中查看

## 故障排除

### 问题：无法录音
**解决方案**：
1. 检查浏览器是否有麦克风权限
2. 系统偏好设置 → 安全性与隐私 → 隐私 → 麦克风
3. 确保浏览器已勾选

### 问题：照片无法上传
**解决方案**：
1. 确保文件是图片格式（JPG、PNG、GIF）
2. 检查文件大小是否过大
3. 尝试刷新页面

### 问题：导出GIF失败
**解决方案**：
1. 确保至少有一张照片
2. 等待处理完成（可能需要几秒钟）
3. 尝试减少帧数或降低图片质量
